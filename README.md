Telegram‑бот, который принимает ссылку на TikTok‑слайдшоу и возвращает готовое видео с оригинальной музыкой.

##  Возможности

- Принимает ссылку на TikTok (формата `https://vt.tiktok.com/...`)
- Скачивает все изображения и аудиодорожку с помощью внешнего сервиса
- Конвертирует изображения, выравнивает размеры и собирает их в видео 
- Накладывает исходный звук на слайдшоу 
- Отправляет готовый ролик пользователю в Telegram 
## Технологии - Python 3 
- `aiogram` — Telegram Bot API   
- `requests`, `BeautifulSoup` — HTTP и парсинг HTML 
- `Pillow (PIL)` — работа с изображениями 
- `opencv-python (cv2)` — сборка видео из кадров 
- `ffmpeg-python` + установленный FFmpeg — объединение видео и аудио
- `python-dotenv` — загрузка токена бота из `.env` 
## Структура проекта

```
tiktok-slideshow-bot/  
├── main.py # Точка входа, логика бота, FSM, обработка ссылок  
├── videomaker.py # Сборка видео: ресайз изображений, создание ролика, склейка с аудио  
├── download_content.py # Загрузка изображений и аудио с TikTok через snaptik.net  
├── requirements.txt # Зависимости проекта  
├── .env  
└── README.md
```

Директории с контентом создаются динамически, по `id` пользователя:

```
content<user_id>/  
├── img/ # Оригинальные изображения (.webp) из TikTok  
├── converted_img/ # Конвертированные и выровненные по размеру .png  
├── music.mp3 # Аудиодорожка  
├── video.mp4 # Видео без звука  
└── output.mp4 # Финальное видео с аудио
```


##  Установка и запуск

1. Клонировать репозиторий
2. Установить зависимости
3. Установить FFmpeg
4. Создать .env по пример и указать в нем: TOKEN=токен
